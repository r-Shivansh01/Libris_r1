# 📦 books-api

This is the core service for the Books API.

## 📁 Directory Structure
- `src/`: Application source code (Express, Routes)
- `tests/`: Jest and Supertest suites
- `Dockerfile`: Multi-stage production build configuration
- `.eslintrc.json`: Code quality rules

## 🚀 Quick Run
```bash
npm install
npm test
npm start
```

For the full repository overview, including CI/CD and deployment details, please refer to the [root README](../README.md).
